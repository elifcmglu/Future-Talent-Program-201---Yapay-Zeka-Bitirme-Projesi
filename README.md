# Campus Eating

Yemekhane menu uygulamasi (MVP) - ziyaretci ve yonetici ekranlari ile gelistirilmistir.

## Ozellikler

- Ziyaretci ekraninda bugunun menusu, kalori ve alerjen bilgileri
- Tarih gezgini ile farkli gun menulerini goruntuleme
- Yonetici girisi ve menu kaydetme formu
- Firebase Firestore entegrasyonu (opsiyonel), localStorage fallback
- Vitest + Testing Library ile temel testler
- GitHub Actions CI (test + build)

## Teknolojiler

- React + Vite + TypeScript
- Tailwind CSS
- Firebase Firestore
- Vitest + Testing Library

## Kurulum

```bash
npm install
npm run dev
```

## Test ve Build

```bash
npm run test
npm run build
```
